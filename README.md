

Here is a look at what I built under the hood:

🔹 RESTful API Architecture: Engineered clean, secure routes to handle user authentication, inventory management, and order processing using structured Mongoose schemas.
🔹 Dynamic Admin Dashboard: Designed a sleek, glassmorphic frontend interface using HTML, CSS, and Vanilla JavaScript. It connects directly to the API via the Fetch API, allowing administrators to perform full CRUD operations—adding new inventory, dynamically modifying product details via custom modals, and tracking customer data in real time.
🔹 Cloud Deployment & Security: Successfully deployed the live server on Render. This project reinforced best practices for using environment variables to protect database credentials, handling CORS, and writing relative paths for cross-environment compatibility.

One of the biggest wins during this build was troubleshooting real-world database connectivity. I learned how to navigate strict network firewalls, bypass DNS SRV lookup blocks using legacy connection strings, and configure IP whitelisting to ensure secure database access.
